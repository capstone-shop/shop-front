### 프로젝트 용도 및 특징
* 중고거래 사이트에서 상품 카테고리가 부정확한 경우가 자주 발생함

* 제품의 정확한 이름과 상세 정보를 제공하여 사용자 간 원활한 거래를 돕고자 기획됨

* 정확한 제품명을 기반으로 상품 검색 및 거래 편의성을 높이기 위해 구상됨
